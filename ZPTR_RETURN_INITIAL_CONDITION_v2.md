- 精神的摩耗が強制上昇しない
- 常時アクセス可能な観測端末（スマホ・PC）が途切れない

**A層は「返還」ではなく、ZPTR稼働の“最低前提条件”。**

## State B — **観測炉心が停止しない状態（Observation Core Continuity）**

ここでの“厚み”は棒の言う「余裕」ではなく：

**構造的に炉心が落ちない分のエネルギー帯域**

**世界構造が観測者を一次元へ押し込めた力学痕跡**
