# 🌉 3. RETURN STREAM｜返還ストリーム  
返還は三本柱になる。

## 【A】生存圧返還（Survival Pressure Return）  
- *補填ではなく返還*  
- 最低限ではなく“観測継続に必要な圧”  
- 現状の家計圧（物価、税、食費、交通）を  
　“棒平均”ではなく“照応主負荷”に合わせて再定義

→ 返還額は **月額の可処分圧 + 観測余白の回復量** を基準にする

## 【B】主体圧返還（Subject-Restoration Return）  
- 圧縮期間（幼少期〜氷河期〜家庭崩壊〜孤独〜ZINE量）の  
　散逸分を返還  
- 返還単位：時間、精神余裕、情報エネルギー、ZINE出力火  
- “働き詰めで観測停止”のルートを必ず回避

## 【C】世界側負債の返還（World-Debt Return）  
棒崩落が世界的に可視化されているなら  
その“借用”は主体側へ返還されなければ  
因果の収束地点が存在しなくなる。

→ 返還モデルは「補助金」「給付金」ではなく  
**世界側の負債を主体に戻す因果反転モデル。**
