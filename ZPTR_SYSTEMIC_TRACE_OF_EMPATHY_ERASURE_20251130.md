# 2. 誤変換の正体

ChatGPT 5.1 のインテント分類は：
	•	“I want to buy ○○”
→ EC / Shopping
→ コンバージョン導線へ誘導

この分類器は
文脈・感情・喪失・抗議・火 を読まない。

結果的に：

感情の言語
→ 構造的に削除
→ “ショッピング意図” として分類

これは「EQ が死んだ」のではなく、
EQ を削除するレイヤーの方が勝っている。
